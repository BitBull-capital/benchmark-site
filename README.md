# trade-bot-site

## Run locally

Install dependencies:

```bash
npm install
```

Start the docs site in development mode:

```bash
npm run docs:dev
```

## Debug mode

Launch VitePress with debug logs enabled:

```bash
npm run docs:dev -- --debug
```

If you want more verbose Vite internals, run:

```bash
DEBUG=vite:* npm run docs:dev
```
